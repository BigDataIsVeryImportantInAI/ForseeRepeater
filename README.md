# audio guiding device for visually handicapped repeaters

----
## 음성유도기, 음향신호기란?

>* 신호등 및 건물 내에 의무적으로 설치되어 장애인과 배려가 필요한 사람들에게 위치와 길을 음성으로 안내하도록 하는 기기

----
## 문제발생

>* 사람들의 관심 밖 -> 잦은 고장

>* 리모콘 소지 필요(잦은 고장, 분실)

>* 1990년대 이후로 발전 없는 노후화된 기술

----
## 니즈 발생

>* 시각장애인의 스마트폰 사용율 및 보급률 급증(2013년 통계 20%↑ -> 2015년 40%↑예상)

>* 리모콘의 잦은 고장을 스마트폰으로 해결하고자 하는 욕구

>* 음향신호기, 음성유도기에 대한 일반인들의 무관심으로 인한 잦은 고장 방치

----
## 개발 목표

>* 시각장애인들이 스마트폰으로 음성유도기를 작동 시킬 수 있도록 해 이동 편의성 향상

>* 기존의 기술과 어우러지며, 최신 기술을 더해서 시각장애인들이 편리한 쪽으로 개인별 커스터마이징 제공

>* 기존 음성유도기 사용 + 스마트폰 사용

----
## 개발 사항
>* 스마트폰으로 음성유도기 및 음향신호기를 작동

>* 음성유도기 및 음향신호기 선택작동(기기 선택 후 가고자 하는 방향의 기기 Contact) 블루투스 4.0을 동작시킬 수 있도록 GATT서버와 접속해 해당 UUID ? Characteristic을 확인하고 특정 신호를 보냄

>* 음성유도기 및 음향신호기를 작동시키지 않고도 스마트폰으로 현재위치 청취가능

>* 시각장애인들이 사용하기 편하도록 화면이 아닌 물리적인 버튼, 음량조절버튼을 통해 어플리케이션이 작동할 수 있도록 하기위해 BroadCast Receiver를 등록

>* 스마트폰으로 현재위치 상세청취가능(주변 지리정보, 운영시간, 기타 등등)

>* 안내요원 호출시 자동으로 전화연결 및 문자메세지 송신

>* 모든 동작과 안내사항은 TTS를 통해 음성으로 송출되도록 함

----
